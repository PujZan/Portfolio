Fruits and Vegetable Freshness Status Detection

Fruits and Vegetables Freshness Status Detection is focused on solving the fruit and vegetable problem that the consumer and seller face by using CNN technique and You Only Look Once (YOLO) to produce most efficient result as possible. This project can massively help in developing the system that the general public most require. It tackles any upcoming issue and also checks the feasibility of the system. This application shall provide a trust worthy environment for buying fruits/vegetables as well as selling them.

Objective:

To develop application for fruit and vegetable freshness detection.

Features:

Input Image

Image Preprocessing

Classification

Machine Learning Model

User Interface

Dataset Handling
